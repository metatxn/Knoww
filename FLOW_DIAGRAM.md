## Key Features

### 1. Dynamic Home Page

- **Default View**: Trending events (highest volume)
- **Breaking**: Recent high-volume events (last 7 days)
- **New**: Today's new events
- **All Categories**: Browse by topic (Politics, Sports, Finance, etc.)
- **Instant Switching**: All data pre-fetched, no loading delays

### 2. Unified Navigation

- **Navbar**: Consistent across all pages
- **Category Links**: Direct access to Politics, Sports, Finance, etc.
- **Wallet Integration**: Connect wallet, view balance, see network
- **Account Dropdown**: Address, balance, disconnect

### 3. Smart Routing

- **`router.back()`**: All back buttons use browser history
- **No hardcoded routes**: Prevents 404 errors
- **Clean URLs**: `/events/[tag]`, `/events/detail/[id]`, `/markets/[id]`

### 4. Performance

- **Parallel Data Fetching**: All event types load simultaneously
- **TanStack Query Caching**: 1-minute stale time
- **Optimized API Calls**: Direct ID-based fetching where possible
- **Lazy Loading**: Categories only load when needed

### 5. User Experience

- **Framer Motion Animations**: Smooth transitions and hover effects
- **Loading Skeletons**: Proper loading states
- **Error Handling**: Graceful error displays
- **Responsive Design**: Mobile, tablet, and desktop layouts

---

**Architecture**: Next.js + Reown AppKit + Polymarket Gamma API + TanStack Query + Shadcn UI + Framer Motion
